## 📖 关于项目

星歌计划文档站是基于 VitePress 构建的现代化文档网站，旨在为 Minecraft 服务器玩家提供：

- 📋 **完整的服务器规则** - 详细的游戏规则和行为准则
- 🎯 **新手指南** - 帮助新玩家快速上手
- 🔧 **插件说明** - 服务器自定义功能介绍
- 📊 **更新日志** - 及时的版本更新信息
- 💡 **游戏技巧** - 实用的游戏攻略和建议

## 🚀 快速开始

### 本地开发

```bash
# 克隆项目
git clone https://github.com/DianFenMian/cdn_starsong_docs.git

# 进入项目目录
cd cdn_starsong_docs

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建项目
npm run build
```
